### playbooks

`vi playbook.yml`

```
- hosts: all
  tasks:
  - name: test connection
    ping:
    
- hosts: ubuntu
  tasks:
  - name: test script
    script: test_touch.sh
    
  - name: test shell
    shell: ls
    register: ps
    
  - debug: msg={{ ps.stdout_lines }}
```

`vi test_touch.sh`

```
touch test_touch.txt
```

```
ansible-playbook playbook.yml -vv
```
