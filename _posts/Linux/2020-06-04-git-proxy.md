This article shows how to enable Proxy when using Git.

If you have enabled proxy like V2Ray on your local machine, you can use this command to set Proxy for Git:

```
git config --global http.proxy localhost:1081
```

Then when you do `git clone`, use the HTTP link rather than the SSH link:
```
 git clone https://github.com/songzy12/xxx.git
```
