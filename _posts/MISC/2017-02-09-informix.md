## setup_tc_db.sh

```
#!/bin/bash

# create dbspace
touch /home/informix/data/spaces/datadbs.000
chmod 660 /home/informix/data/spaces/datadbs.000
onspaces -c -d datadbs -p /home/informix/data/spaces/datadbs.000 -o 0 -s 1240000


cd /home/informix/tc-database-scripts
/opt/apache-ant-1.9.8/bin/ant setup_db
```
