### TightVNC Viewer
After starting VNC service, we can use VNC viewer. The IP address of server can be got from the notification area of task bar. Not sure about what happens when entering 127.0.0.1.

### TightVNC Java Viewer

[Here](https://www.youtube.com/watch?v=kf64OgNokqI) is how to set the security levels through the Java Control Panel. Remember to include also the port number in the exception site list, like: `127.0.0.1:5800`.

Modify line 986 of `utils/websockify/websockify/websocket.py`.

```
# Allow override of signals
original_signals = {
    signal.SIGINT: signal.getsignal(signal.SIGINT),
    signal.SIGTERM: signal.getsignal(signal.SIGTERM),
    #signal.SIGCHLD: signal.getsignal(signal.SIGCHLD),
}
if getattr(signal, 'SIGCHLD', None) is not None:
    original_signals[signal.SIGCHLD] = signal.getsignal(signal.SIGCHLD),

signal.signal(signal.SIGINT, self.do_SIGINT)
signal.signal(signal.SIGTERM, self.do_SIGTERM)

if getattr(signal, 'SIGCHLD', None) is not None:
    if not multiprocessing:
        # os.fork() (python 2.4) child reaper
        signal.signal(signal.SIGCHLD, self.fallback_SIGCHLD)
    else:
        # make sure that _cleanup is called when children die
        # by calling active_children on SIGCHLD
        signal.signal(signal.SIGCHLD, self.multiprocessing_SIGCHLD)
```
