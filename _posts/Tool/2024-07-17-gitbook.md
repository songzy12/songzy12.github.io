Basic usage of GitBook.

## Install

<https://github.com/nvm-sh/nvm>

```
brew install nvm
```

```
nvm install 12
nvm use 12
npm install graceful-fs@4.2.0 -g
npm install gitbook-cli@2.1.2 -g
```

## Run

```
gitbook init
```

```
gitbook serve
```
