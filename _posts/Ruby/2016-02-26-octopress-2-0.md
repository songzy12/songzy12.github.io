## Setup

```
gem install bundler
bundle install
bundle exec rake gen_deploy
```

## Usage

`rake deploy` will push the content of directory `_deploy/` to remote repository `origin`.

Then we push all the content of `source` branch to remote repository `octopress`.  

Note we ignore `_deploy` in `.gitignore`.

```
$ git remote -v
origin  git@github.com:songzy12/songzy12.github.io.git (fetch)
origin  git@github.com:songzy12/songzy12.github.io.git (push)
$ git add .
$ git commit -m 'source update'
$ git push octopress
```

```
$ cd _deploy
$ git init
$ git remote add octopress https://github.com/songzy12/octopress.git
$ cd ..
$ rake gen_deploy
```

## rake

```
rake setup_github_pages
```

Setup a master branch in the `_deploy` directory for deployment.

```
rake new_post['Octopress']
```

Create `source/_posts/2011-07-03-octopress.markdown`.

```
rake generate
```

Generates posts and pages into the `public` directory.

```
rake watch
```

Watches `source/` and `sass/` for changes and regenerates.

```
rake preview
```

Watches, and mounts a webserver at `http://localhost:4000`.

```
rake gen_deploy
```

This will generate your blog, copy the generated files into `_deploy/`, add them to git, commit and push them up to the master branch. Use `Git Shell`.

## _deploy

`robots.txt`（统一小写）是一种存放于网站根目录下的ASCII编码的文本文件，它通常告诉网络搜索引擎的漫游器（又称网络蜘蛛），此网站中的哪些内容是不应被搜索引擎的漫游器获取的，哪些是可以被漫游器获取的。

允许所有的机器人：

```
User-agent: *
Disallow:
```

`Sitemap`指令被几大搜索引擎支持（包括Google、Yahoo、Bing和Ask），指定了网站`Sitemaps`文件的位置。 `Sitemaps`文件包含了网站页面所在的URL的一个列表。

```
Sitemap: <http://www.example.com/sitemap.xml>
```

```
jekyll build

jekyll 2.5.3 | Error:  undefined method `[]' for nil:NilClass
```

It is caused by `highlighter`. See `_config.yml`:

```
markdown: rdiscount
rdiscount:
  extensions:
    - autolink
    - footnotes
    - smart
highlighter: pygments
```

Though Github flavored markdown uses `redcarpet2`, this markdown parser is not supported by `Jekyll` by default.

Modifying hosts can make the accesss of Github faster, due to the reason of `googleapi`.
