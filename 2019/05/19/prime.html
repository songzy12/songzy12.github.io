设三个质数是 x, y, z

则 $\frac{1}{x} + \frac{1}{y} + \frac{1}{z} = \frac{xy+yz+zx}{xyz} = \frac{671}{1022}$

因为上面这个式子是不存在约分的情况，

所以我们知道 $xy+yz+zx = 671​$, $xyz = 1022​$

我们可以通过 xyz = 1022 分解质因数来得出 1022 = 2 * 511 = 2 773

所以三个质数的和是 2+7+73 = 82

两点说明：1. 为什么上面那个式子不存在约分的情况 2. 如何分解质因数

上面那个式子不存在约分的情况是因为

分母 xyz 的因子只有 x, y, z 三个数

而对于分子来说，以因子 x 为例：

(xy + yz+ zx) 其中 xy, zx 均能整除 x, 而 yz 不能整除 x （因为它们三个都是质数）

所以 (xy + yz + zx) 不能整除 x； x 不是 xy + yz + zx 的因子

同理 y, z 也不是。

如何分解质因数：首先我们能看出 2 是一个因子，1022 = 2 * 511

如何分解 511: 首先 $\sqrt{511} < 25$ , 因为 25 * 25 > 511

所以我们只要尝试 25 以内的所有素因数就可以了：2,3,5,7,11,13,17,19,23

当我们试到 7 的时候发现能整除，511 / 7 = 73

为什么我们只要尝试 $\sqrt{511}$ 以内的素因数就可以了呢？

一个结论：如果一个数 n 含有素因子，则其最小的那个素因子一定不超过 $\sqrt{n}$

我们可以使用反证法，假设最小的那个素因子 $> \sqrt{n}$, 则最小的两个素因子乘起来就 $>n$ 了。

好的我们看到 x = 2，然后有

$yz = 511$

$2 (y+z) + yz = 671$

于是 $y + z = 80$

从而 $511/z + z = 80$, $z2 - 80z + 511 = 0$

套公式

$z = (80 + \sqrt{80*80 - 4 * 511}) / 2 = $ (80 + 66) / 2 = 73

$y = (80 - 66) / 2 = 7$
