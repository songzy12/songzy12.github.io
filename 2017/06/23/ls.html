ls

  • -F(–classify): append indicator (one of */=>@|) to entries

  • -1: list 1 file per line

  • / is a directory

  • @ is a symlink
  • | is a named pipe (fifo)
  • = is a socket.
  • * for executable files
  • > is for a “door” – a file type currently not implemented for Linux, but supported on Sun/Solaris.
