虚拟桌面

Win 10

  • 切換這個桌面中的工作視窗： Alt + Tab
  • 呼叫虛擬桌面，展開多桌面工作管理： Win + Tab
  • 快速切換虛擬桌面視野： Win + Ctrl + 左右
  • 快速新增與關閉虛擬桌面： Win + Ctrl + D ， Win + Ctrl + F4
  • 快速分割不同畫面： Win + 上下左右

Ubuntu

  • 快速切換虛擬桌面視野： Alt + Ctrl + 上下左右
  • 展開多桌面工作管理： Alt + Ctrl + Shift + 上下左右

PostMessage

Each message consists of a message identifier and two parameters, wParam(16-bit word) and lParam(32-bit long).

BOOL CALLBACK EnumWindowsProc(HWND hwnd,LPARAM lPram) {
    PostMessage(hwnd,WM_CLOSE,0,0);
    return (true);
}

EnumWindows

EnumWindows(EnumWindowsProc, 0);

In lparam, you can pass a simple variable to the EnumProc method.
