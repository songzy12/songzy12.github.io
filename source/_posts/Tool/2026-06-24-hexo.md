Basic usage of Hexo.

## Quick Start

### Create a new post

``` bash
$ hexo new post "My New Post"
```

More info: [Writing](https://hexo.io/docs/writing.html)

### Run server

``` bash
$ hexo server
```

More info: [Server](https://hexo.io/docs/server.html)
