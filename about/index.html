Internship Experience

Microsoft STCA (2018 Summer)

The Click Through Rate Prediction Module.

  • We apply the Wide & Deep model to the CTR prediction task using TensorFlow.
  • Four types of features are used: article, user, context, and combined features.
  • The AUC is about 0.89 on our dataset, about 0.05 higher than FM model.

XuetangX (2017 Spring)

Question Answering System on MOOC Platform.

  • The question classification module to classify the user query into one of the predefined categories. We use SVM model together with n-gram TF-IDF features. The F1 score is 0.77 on our test dataset.
  • The web spider to collect candidate question-answer pairs from community QA site like zhihu.com.
  • The question matching module to match the user query to one of our candidate question-answer pairs. We run experiments with the MatchZoo toolkit, the precision is about 0.84.
  • This system is running online on http://www.xuetangx.com.

  • API: Image CAPTCHA, SMS Code, Activity List, Activity Detail
  • The profile card recommendation algorithm.

LECP (2017 Spring)

Algorithm Demo of Largest Empty Convex Polygon Problem.

  • Given a set of points in 2-dimensional space, how to find the empty convex polygon with most points.
  • We use duality to get an angular sort of all points in time O(n log n) rather than O(n2 log n).
  • The code is available at https://github.com/songzy12/LECP.
